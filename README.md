## Bambu Lab 프린터 펌웨어 저장소

**현재 지원하는 디바이스**
- P1P
- P1S
- A1 MINI
- A1
- X1C
- X1 (정보 필요)

**정보 제공이 필요한 경우**:

1. 정보 제공 방법 섹션을 참고하여 부탁드립니다.
2. 가끔은 최신 버전이 아닐 수 있습니다. 필자는 P1P를 사용하고 있어서 P1P/S 프린터 펌웨어를 확인할 수 있지만 다른 디바이스에 대해서는 확인이 어렵습니다. 이 경우 해당 디바이스를 사용 중인 사용자가 정보를 제공할 수 있습니다.
2. Bambu Lab 계정 정보가 필요합니다. 또한 계정에 해당 디바이스가 바인딩되어 있어야 합니다. 보통은 글로벌 사용자분들께서 제공이 필요합니다.

### 정보 제공 방법
- [Printer Firmware Provisioning](https://web-user-bambulab-firmware-frontend-5b8v2nlsa3msrp.sel5.cloudtype.app/)  사이트를 방문하여 Bambu Lab 프린터 시리얼 번호를 "Printer Serial Number" 필드에 입력합니다. 그런 다음 Bambu Lab 계정 정보를 입력합니다. (내수 계정도 가능합니다. 중국 번호 입력)

- "Login" 버튼을 클릭합니다. 해당 디바이스의 최신 펌웨어 정보가 표시됩니다. `Realse Note`

### 저장소 업데이트
- "Repo Update" 버튼을 클릭하여 저장소를 업데이트할 수 있습니다. 팝업이 표시되면 한 번 더 확인을 요청합니다. "Yes"를 선택하여 업데이트합니다.

-  이후 성공 메시지가 표시됩니다.
1. "No files to update. 00.00.00.00"은 이미 저장소의 해당 프린터 펌웨어 주소가 최신임을 의미합니다.
2. "File updated successfully. Thank you."는 저장소의 해당 프린터 이전 펌웨어 주소가 최신으로 업데이트되었음을 의미합니다.
